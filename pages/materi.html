🏗️ HTML Dasar

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur konten di halaman web. HTML menyediakan elemen-elemen yang memberitahu browser bagaimana menampilkan konten.

Apa itu HTML?

HTML adalah fondasi dari setiap halaman web. Dengan HTML, kita mendefinisikan elemen-elemen seperti paragraf, heading, link, gambar, dan banyak lagi.

Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur dasar yang sama:

<!DOCTYPE html> <html lang="id"> <head> <meta charset="UTF-8"> <title>Judul Halaman</title> </head> <body> <!-- Konten halaman --> </body> </html>

Tag-Tag Umum HTML

  • <h1> - <h6>: Heading (judul) dari level 1 hingga 6
  • <p>: Paragraf teks
  • <a>: Link/tautan
  • <img>: Gambar
  • <ul> <ol>: Daftar tidak terurut dan terurut
  • <div>: Container/pembungkus konten
  • <button>: Tombol interaktif
  • <form>: Formulir input data

Contoh Penggunaan Tag HTML

<h1>Selamat Datang</h1> <p>Ini adalah paragraf pertama saya.</p> <a href="https://example.com">Klik di sini</a> <img src="gambar.jpg" alt="Deskripsi Gambar">

🎨 CSS Dasar

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur gaya dan tata letak elemen-elemen HTML. CSS memungkinkan kita membuat halaman web yang lebih menarik dan responsif.

Apa itu CSS?

CSS mengontrol bagaimana elemen HTML ditampilkan. Dengan CSS, kita dapat mengatur warna, ukuran, posisi, dan banyak aspek visual lainnya.

Tiga Cara Menggunakan CSS

  1. Inline CSS: Langsung di tag HTML
  2. Internal CSS: Di tag <style> dalam <head>
  3. External CSS: File .css terpisah

Contoh Inline CSS

<h1 style="color: blue; font-size: 24px;"> Judul Berwarna Biru </h1>

Contoh Internal CSS

<style> h1 { color: blue; font-size: 24px; } p { color: gray; line-height: 1.6; } </style>

Properti CSS Umum

  • color: Warna teks
  • background-color: Warna latar belakang
  • font-size: Ukuran font
  • margin: Jarak luar elemen
  • padding: Jarak dalam elemen
  • width dan height: Lebar dan tinggi
  • border: Garis tepi
  • text-align: Perataan teks

Contoh Styling Elemen

.card { background-color: white; border: 1px solid #ddd; padding: 20px; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); }

💻 Coding Dasar (JavaScript)

JavaScript adalah bahasa pemrograman yang memungkinkan halaman web menjadi interaktif. Dengan JavaScript, kita dapat merespons tindakan pengguna dan memodifikasi halaman secara dinamis.

Apa itu JavaScript?

JavaScript dijalankan di browser pengguna, memungkinkan halaman web merespons input pengguna secara real-time. JavaScript adalah inti dari interaktivitas web modern.

Cara Menambahkan JavaScript

  1. Inline JavaScript di dalam tag HTML
  2. Internal JavaScript di tag <script>
  3. External JavaScript di file .js terpisah

Contoh Variabel dan Tipe Data

// Deklarasi variabel let nama = "Budi"; // String let usia = 16; // Number let aktif = true; // Boolean let hobi = ["Coding", "Gaming", "Baca"]; // Array console.log(nama); // Output: Budi

Contoh Fungsi

function sapa(nama) { return "Halo, " + nama + "!"; } console.log(sapa("Rina")); // Output: Halo, Rina!

Contoh Kondisi (If-Else)

let nilai = 85; if (nilai >= 80) { console.log("Grade A - Excellent!"); } else if (nilai >= 70) { console.log("Grade B - Good"); } else { console.log("Grade C - Pass"); }

Contoh Loop (For)

// Loop sederhana for (let i = 1; i <= 5; i++) { console.log("Perulangan ke-" + i); } // Loop array let buah = ["Apel", "Mangga", "Pisang"]; for (let i = 0; i < buah.length; i++) { console.log(buah[i]); }

Konsep Penting dalam Coding

  • Variabel: Tempat menyimpan data
  • Tipe Data: String, Number, Boolean, Array, Object
  • Fungsi: Blok kode yang dapat digunakan kembali
  • Kondisi: Membuat keputusan berdasarkan kondisi
  • Loop: Mengulang kode berkali-kali
  • Array: Koleksi data dalam satu variabel
  • Object: Koleksi properti dan metode
