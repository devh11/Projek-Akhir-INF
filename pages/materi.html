HTML Dasar: Apa itu Tag dan Struktur Halaman

🤔 Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membuat struktur halaman web. Bayangkan HTML seperti tulang atau kerangka dari sebuah rumah. HTML memberi tahu browser (Chrome, Firefox, Safari) bagian-bagian apa saja yang ada di halaman web kita.

🏷️ Apa itu Tag HTML?

Tag adalah instruksi khusus dalam HTML yang ditulis di antara tanda < dan >. Setiap tag memiliki tujuan yang berbeda.

Contoh tag-tag penting:

  • <h1> - Membuat judul besar
  • <p> - Membuat paragraf (teks biasa)
  • <a> - Membuat hyperlink (tautan)
  • <img> - Menampilkan gambar
  • <button> - Membuat tombol
💡 Tips: Sebagian besar tag HTML harus dibuka dan ditutup. Contoh: <p>Isi teks</p>. Tag penutup selalu diawali dengan garis miring (/).

🏘️ Struktur Dasar HTML

Setiap halaman HTML memiliki struktur standar yang sama:

<!DOCTYPE html> <html lang="id"> <head> <title>Judul Halaman</title> </head> <body> <h1>Selamat Datang!</h1> <p>Ini adalah paragraf pertama saya.</p> </body> </html>

Penjelasan struktur:

  • <!DOCTYPE html> - Memberi tahu browser ini adalah file HTML
  • <html> - Tag pembungkus utama (akar dari semua elemen)
  • <head> - Tempat informasi halaman (judul, link CSS, dll)
  • <body> - Tempat konten yang akan ditampilkan di layar
✅ Ingat: HTML hanya mengatur struktur. Untuk membuat halaman terlihat cantik, kita memerlukan CSS!

CSS Dasar: Memberi Warna dan Mengatur Ukuran Teks

🤔 Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa untuk membuat halaman web terlihat lebih cantik dan menarik. Jika HTML adalah kerangka rumah, maka CSS adalah cat, furnitur, dan dekorasi. CSS mengubah tampilan visual dari elemen HTML.

🎯 3 Cara Menulis CSS

Ada 3 cara untuk menambahkan CSS ke halaman HTML:

  • 1. Internal CSS - Ditulis di dalam tag <style> di file HTML
  • 2. External CSS - Ditulis di file terpisah (file.css)
  • 3. Inline CSS - Ditulis langsung di dalam tag HTML (tidak disarankan)

🎨 Memberi Warna dengan CSS

Untuk memberi warna, gunakan properti color (untuk teks) atau background-color (untuk latar belakang).

/* Mengubah warna teks menjadi biru */ p { color: blue; } /* Mengubah latar belakang menjadi kuning */ body { background-color: yellow; } /* Menggunakan kode warna heksadesimal */ h1 { color: #1a4d7a; }

Cara menulis warna:

  • Nama warna: red, blue, green, dll
  • Heksadesimal: #FF5733 (sistem 6 digit untuk warna yang lebih presisi)
  • RGB: rgb(255, 87, 51) (Red, Green, Blue)

📏 Mengatur Ukuran Teks

Gunakan properti font-size untuk mengubah ukuran teks.

/* Teks besar */ h1 { font-size: 32px; } /* Teks sedang */ p { font-size: 16px; } /* Teks kecil */ .catatan { font-size: 12px; }

Satuan ukuran teks:

  • px - Pixel (ukuran tetap) - paling umum
  • em - Relatif terhadap ukuran font induk
  • rem - Relatif terhadap ukuran font root
  • % - Persentase
💡 Tips Desain: Gunakan ukuran besar untuk judul (32px-48px), sedang untuk subjudul (20px-24px), dan normal untuk teks biasa (14px-16px).

🌈 Contoh Lengkap

<style> h1 { color: #1a4d7a; font-size: 36px; } p { color: #2c3e50; font-size: 16px; } body { background-color: #f8f9fa; } </style>
✅ Ingat: CSS membuat halaman terlihat bagus, tapi HTML tetap diperlukan sebagai struktur dasar!

Coding Dasar: Variabel dan If-Else

🤔 Apa itu Coding/Pemrograman?

Coding adalah proses menulis instruksi untuk komputer agar melakukan sesuatu. Bayangkan Anda memberi perintah kepada robot untuk membuat teh - Anda harus menjelaskan setiap langkah dengan jelas. Begitu juga dengan coding!

📦 Apa itu Variabel?

Variabel adalah tempat penyimpanan data sementara dalam program. Seperti kotak yang dapat menampung nilai atau informasi.

// Membuat variabel let nama = "Budi"; let umur = 16; let nilai = 85.5; // Menggunakan variabel console.log(nama); // Output: Budi console.log(umur); // Output: 16

Jenis-jenis data yang bisa disimpan:

  • String (Teks): "Halo" atau 'Dunia'
  • Number (Angka): 42 atau 3.14
  • Boolean (Benar/Salah): true atau false
💡 Tips Penamaan: Nama variabel sebaiknya deskriptif, misalnya: umurSiswa, hargaBuku, bukan x atau y.

🔀 Apa itu If-Else?

If-Else adalah cara program untuk membuat keputusan berdasarkan kondisi. Jika kondisi benar (true), jalankan kode A. Jika salah (false), jalankan kode B.

Struktur dasar If-Else:

if (kondisi) { // Kode yang dijalankan jika kondisi TRUE } else { // Kode yang dijalankan jika kondisi FALSE }

📌 Contoh Praktis If-Else

Kasus: Memeriksa apakah siswa lulus atau tidak

let nilai = 75; if (nilai >= 70) { console.log("Nilai Anda: LULUS ✓"); } else { console.log("Nilai Anda: TIDAK LULUS ✗"); } // Output: Nilai Anda: LULUS ✓

Penjelasan:

  • Variabel nilai menyimpan angka 75
  • Kondisi nilai >= 70 berarti "apakah nilai lebih besar atau sama dengan 70?"
  • Karena 75 lebih besar dari 70, kondisi TRUE, sehingga pesan "LULUS ✓" akan ditampilkan

🔢 Operator Perbandingan

Operator ini digunakan dalam kondisi If-Else:

  • == - Sama dengan
  • != - Tidak sama dengan
  • > - Lebih besar dari
  • < - Lebih kecil dari
  • >= - Lebih besar atau sama dengan
  • <= - Lebih kecil atau sama dengan

🔗 If-Else Bertingkat (Else If)

Kasus: Mengkonversi nilai angka menjadi grade

let nilai = 85; if (nilai >= 90) { console.log("Grade: A"); } else if (nilai >= 80) { console.log("Grade: B"); } else if (nilai >= 70) { console.log("Grade: C"); } else { console.log("Grade: D"); } // Output: Grade: B

Alur program:

  • Cek: Apakah 85 >= 90? TIDAK
  • Cek: Apakah 85 >= 80? YA ✓ → Output "Grade: B"
  • Berhenti (tidak melanjutkan cek yang lain)
✅ Ingat: If-Else adalah fondasi dari logika pemrograman. Semakin baik Anda memahami if-else, semakin mudah belajar bahasa pemrograman lainnya!

🎯 Rangkuman Coding Dasar

  • Variabel: Tempat menyimpan data
  • If-Else: Membuat keputusan berdasarkan kondisi
  • Operator: Simbol untuk membandingkan atau melakukan operasi
  • Logika: Berpikir step-by-step untuk menyelesaikan masalah
