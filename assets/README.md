# Folder Assets

Folder ini digunakan untuk menyimpan aset-aset website seperti:
- 📸 Gambar (images)
- 🎨 Ikon (icons)
- 🎬 Video (videos)
- 🎵 Audio (audio files)

## Struktur yang Disarankan:

```
assets/
├── images/
│   └── (Tempat menyimpan gambar-gambar)
├── icons/
│   └── (Tempat menyimpan ikon-ikon)
└── logo/
    └── (Tempat menyimpan logo)
```

Anda dapat menambahkan file gambar dan ikon di folder ini sesuai kebutuhan.
